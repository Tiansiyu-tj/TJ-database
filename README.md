# 上海地铁交通系统前端展示
本项目是同济大三数据库课设
**个人觉得这个仓库过于混乱（当初是用来方便小组开发而创立的），main分支里面并非我们小组的最终代码，如果想使用完整前后端功能，可以参考xyk分支，或参考链接https://github.com/Tiansiyu-tj/TJU-Database-Project-Assignment/tree/main，里面包含本次课设完整内容**
基于 Vue3 开发的上海地铁交通系统数据库应用前端展示页面，包含地铁流量分析、嘉定校区智能出行决策、地铁导航等功能。

## 功能特性

- 🚇 用户登录系统
- 📊 实时地铁流量数据展示（Dashboard）
- 🎯 嘉定校区智能出行决策（11号线 vs 14号线推荐）
- 🚌 校区出行时刻表与倒计时（DZ1、北安跨线、822、教职班车）
- 🗺️ 地铁线路图与路径规划
- 📈 多种数据可视化图表（折线图、柱状图、饼图、面积图、热力图）
- 📱 响应式设计，支持移动端和桌面端
- 🎨 现代化的 UI 设计

## 技术栈

- Vue 3
- Vue Router 4
- Element Plus
- ECharts 5
- Vite

## 快速开始

## 页面说明

### 登录页面
- 美观的渐变背景设计
- 表单验证
- 支持回车键登录

### 流量展示页面（Dashboard）
- **统计卡片**：展示今日客流总量、实时在线、运营线路、运营站点
- **实时流量趋势**：显示过去24小时的进站和出站流量趋势
- **站点流量排名**：TOP 10 站点流量排行
- **线路流量分布**：各线路流量占比饼图
- **时段流量分析**：工作日和周末的时段流量对比
- **天气信息展示**：温度、降雨量、风速等实时数据
- **流量分类分析**：通勤流量(C)、居家流量(HB)、非居家流量(NHB)
- **OD流量分析**：热门起终点站流量统计
- **进站/出站流量分类**：各站点的分类流量统计
- **线路详情表格**：所有地铁线路的详细信息，支持排序和查看详情
- **导航按钮**：可跳转到嘉定推荐页面和地铁导航页面

### 嘉定校区智能出行决策页面（Commute）
- **智能推荐**：对比 11 号线安亭 vs 14 号线封浜两条路径，自动推荐最优方案
- **决策依据**：综合考虑时刻表、拥挤度、等待时间、步行距离等因素
- **下一班车倒计时**：实时显示 DZ1、北安跨线、822、教职班车的下一班发车时间
- **拥挤度曲线**：11/14 号线全天拥挤度可视化对比
- **校区出行时刻表**：
  - 支持切换工作日/双休日
  - 支持切换不同线路（DZ1、北安跨线、822、教职班车）
  - 已过班次红色标记，未到班次绿色标记
- **终点选择**：支持选择不同线路和站点作为目的地，预计耗时自动更新
- **导航栏**：可快速跳转到管理员页和地铁线路图页面

### 地铁导航页面（RoutePlanner）
- **静态地铁线路图**：展示完整的上海轨道交通网络示意图
- **地图缩放功能**：支持鼠标滚轮缩放查看
- **路径规划**：
  - 选择起始站点和终点站点
  - 支持时间最短/换乘最少两种策略
  - 显示详细换乘方案和预计时间
- **站点信息**：显示站点所属线路、换乘信息等
- **导航栏**：可快速跳转到管理员页和嘉定推荐页面

## 注意事项

- **数据说明**：
  - Dashboard 页面：所有流量数据为模拟数据，用于展示效果
  - Commute 页面：时刻表数据来自图片数字化，拥挤度数据为模拟曲线
  - RoutePlanner 页面：站点和邻接关系为简化网络，仅用于演示路径规划算法
- **实时更新**：Dashboard 实时数据每30秒自动更新一次
- **登录状态**：保存在 localStorage 中
- **后端对接**：当前前端数据为写死/mock，后续需要对接后端 API，详见 `backend-api-spec.md`（如有）
